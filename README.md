# Xylophone App - Flutter

Welcome to the Xylophone App! This is a simple and interactive mobile app built using **Flutter** that allows you to play xylophone-like notes by tapping on colorful buttons. Each button triggers a sound when pressed, providing a fun way to interact with musical notes.

## Features
- 7 colorful buttons, each corresponding to a different musical note.
- Plays a sound when you tap any button.
- Beautiful UI with a black background to make the buttons stand out.
- Built with **Flutter** and **Assets Audio Player** package to play audio files.
